THAT ONE EXPERIENCE

THE PROBLEM

Back then, there are some people that always tested my peace and autonomy. It reached to the point that whenever I saw them, I felt stressed and terrified. However, I kept saying 'yes' or go with them because I don't want to lose them.

THE SOLUTION

Then, when I decided to say 'No' frequently and stay away from them as much as I can, I felt freedom.

THE LESSON

Then, I realized that I need to set boundaries with other people because that's the way to attain peace and autonomy. Now, it influenced me that if someone has become an arch nemesis to my peace, then I'll assert my boundaries and get away as much as I can.

ETHICAL DILEMMA

THE PROBLEM

You are developing software for a client. While working on the project, you find a copyrighted source code online that perfectly solves the problem. The client will never know if you copy and use it without permission. What would you do?

THE SOLUTION

If I see a copyrighted source code that is useful for my project, the first thing to check is their licensing agreement, specifically, an Open Source License like MIT, BSD 2-Clause, or Apache License 2.0. These licenses allow usage and modification of their code with the need only of crediting them in the LICENSE or NOTICE file.

THE VALUE

The value that guides me through this decision is integrity to honor, accountability to have responsibility, and respect to these people who worked hard to make those source code. This decision reflects a kind of IT professional where crediting is important when you use your code as they worked tirelessly to make that happen.

FOR ME,
ETHICS IS....

a guide that leads myself into the morally right one and avoids the morally wrong one. Think of it as a parent, who always reminds and reprimands you. Ethics done this by reminding you that every action you do is either right or wrong, and if you do wrong actions, consequences follow like a parent reprimanded their children.

I hope to become someone who has accountability and integrity for every action and decision, empathy to other people, a genuine respect for myself and others, has an autonomy for every personal decision and avoid being dragged by other people’s choices, being assertive to my boundaries, and of course, being able to recognize and reflect which action is correct and wrong. With all of the aforementioned things, I’ll be able to become an ethical person.
