# Ethical Identity Profile

A (defunct) interactive, creative format to explore my personal ethics, moral philosophy, and decision-making frameworks across 7 components. 

A supposed partial requirement for **GE8 (Ethics)**.

Website: https://lloyd-christian.github.io/ethical-indentity-profile/

## ✨ Features

* Neobrutalist UI/UX (I love this design).
* Interactive 3D Flip Card for "Who am I?" section.
* Sticky Layer Scroll ("Magic Runway") that changes its content as the black bar passed through for "That One Experience" and "Ethical Dilemma".

## 🛠️ Tech Stack

* Front-end: HTML + CSS + JavaScript
* Hosting: GitHub Pages
* Version Control: Git + GitHub
